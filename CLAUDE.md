# Project Name: 24h Food Tour

This project uses the Claude Code rule specification. All AI behavior, project structure, design system instructions, and technical context are split into modular rule files for better maintainability.

Please refer to the `.claude/rules/` directory:
- `.claude/rules/tech-stack.mdc`: Technology stack details.
- `.claude/rules/project-structure.mdc`: Strict directory architecture rules.
- `.claude/rules/design-system.mdc`: Clay Design System aesthetic principles and constraints.
- `.claude/rules/ai-workflow.mdc`: AI assistant behavior and component guidelines.
