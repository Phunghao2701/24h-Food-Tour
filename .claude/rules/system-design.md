# System Design Principles

## Core Concepts
- **CAP Theorem**: Consistency, Availability, Partition Tolerance.
- **Caching**: Implement at Edge (CDN) and Browser (SWR/React Query).
- **Scalability**: Horizontal scaling, Stateless components.

## Advanced
- **Queues**: Task offloading for heavy operations.
- **Consistency**: Eventual consistency where applicable.
- **Security**: Principle of Least Privilege.
